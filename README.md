# Fourtitude Assessment

## Description

This project is developed as part of the Fourtitude technical assessment.

### Implemented Features

- Question 1
  - Request validation
  - Partner authentication
  - Base64 password validation
  - SHA256 signature validation

- Question 2
  - Required field validation
  - Invalid Total Amount validation
  - Timestamp expiration validation
  - Unauthorized partner validation

- Question 3
  - Base discount calculation
  - Conditional discount
  - Prime number validation
  - Maximum 20% discount cap

## Technology

- ASP.NET Core (.NET 10)
- C#
- SHA256
- Base64
